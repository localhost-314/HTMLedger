Enable Auto-Save?

Auto-save writes your file to disk a few seconds after every keystroke, so you never lose work from a crash or a forgotten save.

Keep in mind: if the timer fires while you're mid-edit, the incomplete change will be committed to disk — particularly worth noting for config files, structured data, or anything syntax-sensitive.

A Revert button will appear in the toolbar whenever auto-save has drifted from your last manual save (Ctrl+S), letting you recover the last intentional version in one click.
