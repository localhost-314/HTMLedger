## HTMLedger

The full-featured edition powered by Monaco — the same editor engine that drives Visual Studio Code.

- **Monaco Editor** with Emmet, minimap, format-on-save, and full IntelliSense-style editing
- **Multi-tab workflow** with unsaved-change indicators on every tab
- **DMARC aggregate report viewer** — open any DMARC XML and instantly see a visual breakdown, no uploads required
- **Snippet library** — 16 built-in code blocks across HTML, CSS, and JS, plus custom snippets that persist across sessions
- **Find in folder** — grep-style search across your entire workspace with file, line number, and code preview
- **Workspace manager** — card grid view with search, sort, and rename. Back up and restore any workspace on a schedule you set
- **File watcher** — get notified when a file changes on disk while you're editing
- **Settings panel** — font family, font size, tab size, autosave, minimap, and more

## HTMLedger Lite

The ultralight edition built on CodeMirror 6. Fast to open, small to ship, and focused on the essentials.

- **CodeMirror 6** with syntax highlighting for HTML, CSS, JS, JSX, TS, TSX, JSON, XML, SVG, Markdown, and plain text
- **Multi-tab editing** with unsaved-change indicators
- **Workspace manager** — the same card-grid experience as the full edition
- **File search** within your workspace
- **Live HTML preview** panel
- **Portable build** — runs directly from a folder, no installer or admin rights needed
- **Settings panel** — font size, tab size, and theme

## Privacy first

Everything runs on your machine. HTMLedger never uploads your files, never phones home, and never creates an account on your behalf. The DMARC viewer parses XML locally. The workspace backup saves to your own disk. There is no server.
